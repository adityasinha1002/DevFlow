# AI Coding Agent (VS Code Extension)

A simple autonomous coding agent powered by Google Gemini that runs inside VS Code.

## Features
- Generate files from prompts
- Gemini AI integration
- VS Code extension architecture

## Setup

Install dependencies:
npm install
Compile:
npx tsc
Run extension:

Press **F5** inside VS Code.

## Usage

Open Command Palette:
Cmd + Shift + P
Run:
Start AI Coding Agent
